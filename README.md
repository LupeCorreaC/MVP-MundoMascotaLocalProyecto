# 🐾 Mundo Mascota Local

Plataforma web para conectar usuarios con servicios locales para mascotas, como veterinaria, peluquería, guardería, transporte y acompañamiento.

---

## 🚀 Funcionalidades

- Registro e inicio de sesión de usuarios.
- Visualización de servicios por categorías.
- Selección de comercios y servicios.
- Cálculo automático de precios.
- Agendamiento de citas.
- Visualización de citas en “Mis citas”.
- Interfaz moderna y responsive.
